The R\+Inside package provides a few classes for seamless embedding of \href{https://www.r-project.org}{\texttt{ R}} inside of C++ applications by relying on \href{http://www.rcpp.org}{\texttt{ Rcpp}}.\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysubsection{Examples}\label{md__r_e_a_d_m_e_autotoc_md2}
Provided with the package itself are nine subdirectories with examples\+: from more than a dozen basic command-\/line examples (in directory {\ttfamily standard}) to graphical user-\/interfaces (using both \href{http://www.qt.io}{\texttt{ Qt}} and \href{http://www.webtoolkit.eu/wt}{\texttt{ Wt}}), linear algebra with \href{http://arma.sf.net}{\texttt{ Armadillo}} and \href{http://eigen.tuxfamily.org/index.php?title=Main_Page}{\texttt{ Eigen}}, parallel computing with M\+PI to a sandboxed server, and (since release 0.\+2.\+16) a simple (and more limited) interface for embedding insice C applications.

The simplest example (modulo its header) is \href{inst/examples/standard/rinside_sample0.cpp}{\texttt{ examples/standard/rinside\+\_\+sample0.\+cpp}}
